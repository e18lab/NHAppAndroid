## Installation

```bash
# 1 Clone repo & install deps
git clone https://github.com/Maks1mio/NHAppAndroid.git
cd NHAppAndroid
npm install

# 2 Run on device / emulator
npm run start
```

### Android build
```bash
npm run android-build
npm run move-apk
```
